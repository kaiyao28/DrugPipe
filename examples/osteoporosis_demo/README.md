# Osteoporosis Demo

The synthetic osteoporosis/BMD/fracture-style demo lives in `data/example/`.

Run it with:

```bash
make example
```

Primary outputs:

```text
results/example/targets/ranked_targets.tsv
reports/example/target_prioritisation_report.md
reports/example/target_cards/
```

The data are synthetic and contain no individual-level information.
